# Анализатор сетевых подключений (NetworkAnalyzer)

## Описание проекта

**NetworkAnalyzer** - это десктопное приложение для анализа сетевых интерфейсов компьютера и проверки доступности веб-ресурсов, разработанное на платформе .NET 8.0 с использованием WPF (Windows Presentation Foundation). Приложение позволяет просматривать информацию о сетевых адаптерах, анализировать URL-адреса и проверять доступность хостов.

### Функциональные возможности

- **Просмотр сетевых интерфейсов**:
  - Отображение всех активных сетевых адаптеров
  - Детальная информация по каждому интерфейсу:
    - IP-адрес и маска подсети
    - MAC-адрес
    - Состояние подключения
    - Скорость соединения
    - Тип интерфейса
  - Визуальный индикатор состояния (зеленый/красный)

- **Анализ URL/URI**:
  - Разбор URL на компоненты:
    - Схема (протокол)
    - Хост
    - Порт
    - Путь
    - Параметры запроса
    - Фрагмент
  - Определение типа адреса (локальный/публичный/loopback)
  - DNS-запрос (получение всех IP-адресов)
  - Ping-проверка доступности хоста

- **История проверок**:
  - Автоматическое сохранение всех проанализированных URL
  - Возможность повторного просмотра результатов
  - Очистка истории

## Технологический стек

- **.NET 8.0** - платформа разработки
- **WPF (Windows Presentation Foundation)** - технология создания интерфейса
- **XAML** - язык разметки интерфейса
- **System.Net.NetworkInformation** - работа с сетевыми интерфейсами
- **System.Net** - работа с DNS и Ping
